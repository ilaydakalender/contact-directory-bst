# Contact Directory Using Binary Search Tree

This project is a Java-based contact directory application that stores contacts alphabetically using a Binary Search Tree (BST).

The application supports core directory operations such as adding, searching, deleting, and displaying contacts through a command-line interface.

## Features
- Add or update a contact
- Search for a contact by name
- Delete a contact
- Display all contacts in alphabetical order

## Data Structure
- Binary Search Tree (BST)
- Recursive insert, search, delete, and traversal operations

## Technologies
- Java
- Scanner for user input

## How to Run
1. Compile all `.java` files.
2. Run the `Main` class.
3. Use the following commands:
   - `ADD name phoneNumber`
   - `SEARCH name`
   - `DELETE name`
   - `SHOW`
   - `EXIT`
